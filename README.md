# Python web Calculator

My second Python project.
Thanks for the frontend to my friend

## Features
- Basic arithmetic: Sum, Sub, Mult, Div, Pow, Root.
- Trigonometry: Sin, Cos, Tan, Cot (input & output in radians).
- Clean modular architecture (Operations, Utils).

## How to run
1. Make sure Python is installed.
2. Download this repository.
3. Run `app.py`.
4. Open your browser and go to `http://localhost:5000`
