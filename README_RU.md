# Веб калькулятор на Python

Мой второй проект на Python.
Спасибо за интерфейс моему другу

## Функции
- Арефметика: Sum, Sub, Mult, Div, Pow, Root.
- Тригонометрия: Sin, Cos, Tan, Cot (ввод и вывод в радианах).
- Чистая модульная архитектура (Operations, utils).

## Как запустить
1. Убедитесь, что установлен Python.
2. Загрузите этот репозиторий.
3. Запустите `app.py`.
4. Откройте свой браузер и перейдите по ссылке `http://localhost:5000`
